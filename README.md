# רובי — בלוקים בעברית (craftOm)

הרחבת MakeCode עבור **Minecraft Education** שמספקת את בלוקי ה**סוכן (Agent)** בעברית,
תואמים אחד-לאחד לבלוקים המקוריים. נבנתה עבור "רובי" — מורה התכנות בתוך המשחק לילדים בגילאי 8–10.

## למה צריך את זה
ב-Minecraft Education אי אפשר להחליף את שפת ה-Code Builder לעברית (Minecraft עצמו לא מתורגם לעברית).
ההרחבה הזו עוקפת את המגבלה: היא מגדירה בלוקים עבריים שעוטפים את אותן פעולות `agent.*` מהמקור,
כך שהילד מתכנת בעברית.

## מה כלול (קטגוריה: "סוכן רובי")
תנועה, סיבוב, הנחה/הריסה/חרישה/תקיפה, איסוף, ניהול מלאי (תאים, השלכה, העברה),
זיהוי ובדיקה בכיוון, וטלפורט אל השחקן.

## מגבלה
בלוקי הליבה של MakeCode — **לולאות, לוגיקה, משתנים, מתמטיקה, טקסט** — מובְנים במנוע
ולא ניתנים לתרגום דרך הרחבה. הם יישארו באנגלית.

## התקנה ב-Code Builder
1. במשחק לחצו **C** לפתיחת Code Builder → MakeCode.
2. גלגל השיניים (⚙️) → **Extensions**.
3. הדביקו את כתובת ה-repo הזה וייבאו.

> Use as Extension — this repository can be added as an **extension** in MakeCode.
